0danbaşla API Dokümantasyonu

Bu API, 0danbaşla uygulaması için RESTful servisler sağlar.

Kullanılabilir Endpointler

GET /api/items

Tüm öğeleri listeler.

GET /api/items/:id

Belirli bir öğeyi ID'ye göre getirir.

POST /api/items

Yeni bir öğe ekler.

{
  "name": "Ürün Adı",
  "description": "Ürün Açıklaması",
  "price": 150
}
PUT /api/items/:id

Bir öğeyi günceller.

{
  "name": "Yeni Ürün Adı",
  "description": "Yeni Ürün Açıklaması",
  "price": 200
}
DELETE /api/items/:id

Bir öğeyi siler.
